# Aegis Mobile Project

**디지털 생존을 위한, 모두를 위한 스마트폰 생태계.**  
Aegis Mobile은 정보, 안전, 생존을 위해 싸우는 시민들을 위한 독립적이고 윤리적인 스마트폰 프로젝트입니다.

## 📦 구성
- 📄 `docs/`: 제안서, 요약자료, NGO/제조사용 문서 등
- 🛠️ `specs/`: 모델별 스펙 문서 (R0 ~ R5)
- 🎨 `images/`: 로고, 디자인 시안
- 🎯 `Pitch-Deck/`: 프레젠테이션 및 외부 공유용 자료
