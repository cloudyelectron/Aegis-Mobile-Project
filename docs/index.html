What is Aegis R0?

Aegis R0 "Rights" is a radically affordable, rugged, and secure mobile device designed for the underserved, activists, journalists, and anyone who needs digital safety in a tough world.

Core Features

  • Price Target: ~$140–150
  • Polycarbonate rugged body, IP69K-rated
  • BasicPort for modular accessories
  • Secure communications, hardware switches
  • Field-ready, long battery life (5000mAh)
  • 12MP rear / 5MP front cameras

Open Collaboration

This project is open-source and aims to be transparent, accessible, and ethical. We welcome help from NGOs, ethical manufacturers, and privacy-first organizations.
